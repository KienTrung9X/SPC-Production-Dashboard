# Chạy SPC Dashboard trên Windows

## Yêu cầu
- Node.js (tải từ https://nodejs.org/)
- Kết nối đến database DB2

## Cách chạy

### Cách 1: Dùng file .bat
```cmd
start.bat
```

### Cách 2: Chạy thủ công
```cmd
npm install
npm start
```

### Cách 3: Chạy server khác
```cmd
node server.js
node simple-server.js
```

## Truy cập
- http://localhost:3000 (simple-server.js)
- http://localhost:3001 (server.js)

## Dừng ứng dụng
Nhấn Ctrl+C trong terminal
